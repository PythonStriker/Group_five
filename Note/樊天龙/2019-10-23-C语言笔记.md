## 一、简介

C语言是为了开发Unix系统研发的编译型语言,主要应用于现代程序,驱动和操作系统,拓展名为.c,生成后的编译文件拓展名为.out。
## 二、开发环境
文本编辑器(vim)和编译器(gcc)
## 三、语法
先编译,后执行。

每个语句必须以分号结束。它表明一个逻辑实体的结束。

#include 预处理器,加载头部文件。

所有的 C 语言程序都需要包含 **main()** 函数。 代码从 **main()** 函数开始执行,main()函数可以为空。

**return 0;** 语句用于表示退出程序。

单行注释://

多行注释:/**/

标准输入输出头文件:<stdio.h>

printf():输出

scanf():输入

gcc如果不指定编译后的文件名,默认为a.out

执行编译文件:./编译文件

指定编译后的文件名:gcc xx.c -o xx.out

变量是内存中一块区域:代码段(编写的程序),数据段(全局变量,常量)，栈(基本数据类型)

名常量是程序执行期间不会改变的量

常量的声明方式:#define(在main函数外声明,没有分号) const

运算符:+ - * / % ++ -- += -= *= /= %=

判断:if if...else if if ...else switch

循环:for while do...while

数组:,索引默认从0开始

声明数组并初始化:

type 数组名[] = {}    (不指定数组元素个数,如果您省略掉了数组的大小，数组的大小则为初始化时元素的个数)

type 数组名[num] = {}    (指定数组元素个数,大括号 { } 之间的值的数目不能大于我们在数组声明时在方括号 [ ] 中指定的元素数目)

指针获取数组中的元素:`int *p =` 数组名(首地址首个元素) 第二个元素:`*(p+1)`,以此类推 

## 四、数据类型

1.基本数据类型:整数类型和浮点类型

2.枚举类型

3.void类型:表示没有返回值

4.派生类型

C语言是强类型语言,声明变量的同时,必须指定数据类型。

## 五、指针

指针就是指向内存中的地址,地址指向的才是存储的东西。

gdb工具是gcc中自带的调试工具。

gcc -g xx.c -o xx.out(加上-g参数才可以调试)

打开gdb工具:gdb ./xx.out

列出程序源代码:list/l(显示不完整,按回车继续执行上一个命令)

开始调试:start

查看堆栈:bt

切换堆栈:f 堆栈名

跳转到下一行:n回车

前面的$1,$2,$3...等是输出第一个第二个第三个变量的意思。

退出:q
